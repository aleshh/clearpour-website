# ClearPour Website

This directory contains a fully static version of the ClearPour website.

Files:

- `index.html` for the homepage
- `support/index.html` for the support page
- `privacy/index.html` for the privacy policy
- `assets/styles.css` for all shared styling

To deploy on DreamHost:

1. Point your domain or subdomain at the directory you want to publish.
2. Upload the contents of this `website` folder to the web root.
3. Make sure HTTPS is enabled.
4. Confirm these URLs load:
   - `/`
   - `/support/`
   - `/privacy/`
